# workflow
Raspberry Pi + Camera → Image Capture with timelapse.py <br>
ImageJ/Fiji → Image Processing and Raw Data Export <br>
Jupyter Notebook → Data Visualization and Analysis with circadian_analysis.ipynb <br>
