## System Components

![System Overview](hardware/picomponents.jpg)

The image above shows all components used in the Raspberry Pi–based circadian imaging system prior to assembly.

- Raspberry Pi 4B
- Raspberry Pi Camera V2 NoIR and case
- 3D Printed Light Ring (holder for the LED diodes)
- LED diodes (white + IR)
- Logic converter
- Power supply
- microSD card
